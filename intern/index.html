My Task

Design a webpage to upload image and view it in VR mode.

  • Upload an image in firebase storage and get downloadable url.
  • Store these url in firebase database.
  • The uploaded image is displayed in VR sphere thumbnail.
  • Created VR sphere dynamically based on number of images uploaded in firebase.
  • Added the addeventlisterner to these VR sphere.
  • When these spheres are clicked 360 image will be displayed.
